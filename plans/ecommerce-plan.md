# Dondad Tech E-Commerce Site

## Overview
A clean, simple single-vendor e-commerce platform for selling phones, laptops, tablets, and accessories.

## Design Philosophy
- Semantic HTML5
- Simple CSS (no frameworks)
- Vanilla JavaScript (no libraries)
- Straightforward, authentic code

---

## File Structure

```
Dondad Tech/
├── index.html           # Homepage
├── shop.html            # All products
├── product.html         # Product detail page
├── cart.html            # Shopping cart
├── checkout.html        # Checkout flow
├── login.html           # Login page
├── register.html        # Register page
├── admin.html           # Admin dashboard
├── style.css            # Main stylesheet
├── script.js            # Main JavaScript
└── images/              # Product images
```

- Product browsing
